# WHO Global Health Dashboard

An interactive web dashboard analysing global health indicators across 193 countries and 16 years using WHO data.

## Features
- Global choropleth map by health indicator
- Country deep dive with trend analysis
- Regional comparison (Developed vs Developing)
- Anomaly detection with z-score alerts
- Correlation explorer with interactive scatter plots

## Tech Stack
Python, Dash, Plotly, Pandas, NumPy, SciPy
